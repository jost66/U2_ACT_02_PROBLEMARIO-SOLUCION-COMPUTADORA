# U2_ACT_02_PROBLEMARIO-SOLUCION-COMPUTADORA
Solución computacional desarrollada para el análisis de fatiga en ejes escalonados, automatizando el cálculo de concentración de esfuerzos y criterios de falla según la metodología de Shigley.
#  Sistema Computacional para el Diseño de Ejes por Fatiga

##  Descripción del Proyecto
Este repositorio contiene el código fuente de un algoritmo desarrollado en Python diseñado para automatizar la resolución de casos prácticos de diseño de ejes escalonados. El sistema evalúa elementos sometidos a cargas combinadas fluctuantes (flexión y torsión) aplicando la metodología de Shigley.

###  Características del Algoritmo
* **Cálculo automático de factores de concentración de esfuerzos por fatiga** ($K_f$ y $K_{fs}$).
* **Análisis de esfuerzos nominales y de Von Mises** (medios, alternantes y máximos).
* **Evaluación simultánea de 4 criterios de falla por fatiga:**
  - Goodman modificado
  - Soderberg
  - Gerber
  - ASME-Elíptica
