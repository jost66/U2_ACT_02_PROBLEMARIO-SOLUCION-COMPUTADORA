# U2_ACT_02_PROBLEMARIO-SOLUCION-COMPUTADORA
Solución computacional desarrollada para el análisis de fatiga en ejes escalonados, automatizando el cálculo de concentración de esfuerzos y criterios de falla según la metodología de Shigley.
# Sistema de Diseño de Ejes por Fatiga (Método Shigley)

## Descripción
Este repositorio contiene el algoritmo desarrollado en Python para resolver casos prácticos de diseño de ejes sometidos a cargas combinadas fluctuantes, evaluando las teorías de falla por fatiga.
